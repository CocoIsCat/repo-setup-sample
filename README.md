# repo-setup-sample

Description/Abstract (프로젝트에 대한 한 줄 요약)
- 이 프로젝트는 Github에서 Repository를 사용할 때 관리해야 할 것들을 연습하기 위한 repo입니다.

데모 버전
- [See Demo](https://github.com/CocoIsCat/repo-setup-sample)

## Prereuisite

개발 환경

- node.js: ^11.0
- java: ^8

## How to Start

실행 방법

```shell
$ git clone {repo addr}
cd repo-setup-sample
npm install
npm start
```
